# Machine Documentation & Workflow Library

This repository is a current, centralized documentation, setup procedures, and workflows for HKS LA_MADE Machines.

## Purpose

* Standardize machine usage and workflows
* Maintain reproducible fabrication processes
* A base for advanced automation and system integration

## Machines

* Laguna CNC Router (61" x 121")
* Laguna Laser Cutter
* Bambu Lab X1E
* Modix BIG-120X
* Standard Bots RO1 (6-axis robot)
* WASP Clay Kit 3.0

## Structure

Each machine includes:

* 01_Hardware documentation
* 02_Setup guides
* 03_Example/Starter files
* 04_Advanced workflows
